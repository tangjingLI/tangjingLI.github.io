#### 括号匹配
<font size=2>&emsp;&emsp;判断括号串是否匹配，遇左括号压栈，遇右括号出栈，检查左右括号是否匹配，若遇右括号时栈为空，或扫描完成后栈不为空，则返回false</font>

```java
    public boolean isValid(String s) {
        if(s.length()==0)
            return true;
        if(s.length() % 2 != 0)     //字符串长度为奇数时肯定不匹配
            return false;

        Stack<Character> stack = new Stack<>();
        for (int i = 0; i < s.length(); i++) {
            char temp = s.charAt(i);
            if (temp == '(' || temp == '[' || temp == '{') {
                //遇左括号压栈
                stack.push(temp);
            } else if (temp == ')' || temp == ']' || temp == '}') {
                //遇右括号出栈
                if(stack.empty())   //栈空时不匹配
                    return false;
                char ch = stack.pop();
                switch (temp) {
                    case ')':
                        if (ch != '(')
                            return false;
                        break;
                    case ']':
                        if (ch != '[')
                            return false;
                        break;
                    case '}':
                        if (ch != '{')
                            return false;
                        break;
                    default:
                        break;
                }
            } else {
                return false;
            }
        }
        if(!stack.empty())      //栈非空时不匹配
            return false;
        return true;
    }
```

#### 最长不重复子串

<font size=2>&emsp;&emsp;给出一个字符串，求它的最长的无重复字符的子串，使用双指针实现</font>

```java
    public void func(String s) {
        ArrayList<Integer> start=new ArrayList<>();
        int maxLen = 1;

        int begin=0,end=0;
        ArrayList<Character> arr=new ArrayList<>();
        while (begin<s.length()&&end<s.length()){
            if(!arr.contains(s.charAt(end))){
                arr.add(s.charAt(end));
                end++;
            }else {
                if(maxLen==(end-begin)){
                    start.add(begin);
                }else if(maxLen<(end-begin)){
                    maxLen=end-begin;
                    start.clear();
                    start.add(begin);
                }
                while (arr.contains(s.charAt(end))){
                    arr.remove(0);
                    begin++;
                }
            }
        }

        //检查字符串末尾
        if(maxLen==(end-begin)){
            start.add(begin);
        }else if(maxLen<(end-begin)){
            maxLen=end-begin;
            start.clear();
            start.add(begin);
        }

        for(int i=0;i<start.size();i++){
            System.out.println(s.substring(start.get(i),start.get(i)+maxLen));
        }
    }
```
